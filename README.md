# 🙏 Ministério Evangélico Valentes de Cristo

> Site institucional desenvolvido para apresentar a igreja, seus cultos, eventos e facilitar o contato com membros e visitantes.

## 📖 Sobre o projeto

Este projeto é um site institucional desenvolvido para o **Ministério Evangélico Valentes de Cristo**, com o objetivo de:

- Apresentar a igreja e sua missão
- Divulgar cultos e eventos
- Facilitar o contato com visitantes
- Fortalecer a presença digital da igreja

## ✨ Funcionalidades

- 📅 Agenda de cultos e eventos
- 📍 Informações da igreja (localização, horários)
- 📞 Contato direto via WhatsApp
- 📱 Layout totalmente responsivo
- 🎨 Design moderno e acolhedor
- ⚡ Carregamento rápido
